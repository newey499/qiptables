00041 NAME
00042        getuid, geteuid - get user identity
00043 
00044 SYNOPSIS
00045        #include <unistd.h>
00046        #include <sys/types.h>
00047 
00048        uid_t getuid(void);
00049        uid_t geteuid(void);
00050 
00051 DESCRIPTION
00052        getuid() returns the real user ID of the calling process.
00053 
00054        geteuid() returns the effective user ID of the calling process.
00055 
00056        When  a  normal program is executed, the effective and real user ID of the process are set to the ID of the
00057        user executing the file.  When a set ID program is executed the real user ID is set to the calling user and
00058        the effective user ID corresponds to the set ID bit on the file being executed.
00059 
00060 ERRORS
00061        These functions are always successful.

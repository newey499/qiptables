Detailed Description

Abstract Class to provide a persistent "Never display this dialog again" dialog box.

The persistency is provided by assigning a key generated in a subclass and using this key to uniquely identify the dialog. The generated key is the name of the subclass

This means that a sub class must be defined for each "Never display this dialog again" dialog box.

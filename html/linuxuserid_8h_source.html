41 NAME
42  getuid, geteuid - get user identity
43 
44 SYNOPSIS
45  #include <unistd.h>
46  #include <sys/types.h>
47 
48  uid_t getuid(void);
49  uid_t geteuid(void);
50 
51 DESCRIPTION
52  getuid() returns the real user ID of the calling process.
53 
54  geteuid() returns the effective user ID of the calling process.
55 
56  When a normal program is executed, the effective and real user ID of the process are set to the ID of the
57  user executing the file. When a set ID program is executed the real user ID is set to the calling user and
58  the effective user ID corresponds to the set ID bit on the file being executed.
59 
60 ERRORS
61  These functions are always successful.
62

QWidget *  parent = 0,
Qt::WindowFlags  f = 0

explicit

Constructor used when form is employed as a seperate window and used to insert snippets into rulesets.

The required behaviour is obtained by a combination of the popSnippetSelect parameter and the Qt::WindowFlags parameter. This means that the pop up window is hidden when the application is closed or the ruleset form is hidden.

Parameters
popupSnippetSelect- true if form is to be displayed as pop up seperate window else false
parent- reference to QT parent Object
f- Window behaviour flag
